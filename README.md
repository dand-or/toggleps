# toggleps

## What is toggleps

This is a zsh script for mac when you want to switch Photoshop language between English and Japanese.

## Usage

1. git clone where you like
2. edit ~/.zshrc
```
alias toggleps="sudo your/path/toggleps/toggleps.sh"
```
3. do source command
```
$ source ~/.zshrc
```

## target
photoshop CC 2021
